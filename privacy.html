プライバシーポリシー・利用規約

本規約は、制作者（nanadqw）が提供する「なかまモンスターバトル シミュレーター」（以下「本アプリ」）におけるユーザーデータの取り扱いを説明するものです。

Google API Services User Data Policy 対応
本アプリは、Google APIから取得した情報を、限定的な使用に関するポリシーを含む Google API Services User Data Policy に準拠して使用および転送します。

1. データの収集とアクセス (Data Accessed)

本アプリは、Google OAuth認証を通じて以下のユーザー情報にアクセスします：

  • メールアドレス (Email address): ユーザーの識別および本アプリへのログイン認証のため。
  • 基本的なプロフィール情報 (Basic profile info): アプリ内での表示名およびプロフィール画像の表示（ユーザーの利便性向上）のため。
  • GoogleユーザーID (OpenID): 各ユーザーのアカウントを一意に識別し、データを安全に紐付けるため。

2. データの利用目的 (Data Usage)

取得したデータは、以下の目的のためにのみ使用され、それ以外の目的（広告への利用や第三者への販売等）には一切使用しません：

  • 本アプリのログイン・認証機能の提供。
  • ユーザーが本アプリ上で作成した「モンスターデータ」および「チーム編成データ」を、本人のアカウントに紐付けて保存・管理し、異なるデバイス間での同期を可能にするため。
  • 取得した個人データを第三者に販売、共有、または広告目的で利用することはありません。
  • 本アプリの利便性向上のため、個人を特定できない形での統計データ（例：モンスターの使用率ランキング等）の作成および公開に利用することがあります。

3. データの保存と保護 (Data Storage)

取得した情報は、適切なセキュリティ対策を講じたデータベース（Supabase）にて厳重に管理されます。ユーザーは、本アプリからログアウトすることでセッションを終了でき、また必要に応じて保存データの削除を要求することができます。

4. 免責事項

本アプリは「ドラゴンクエストウォーク」の非公式ファンツールであり、株式会社スクウェア・エニックスおよび株式会社ゲームフリーク等とは一切関係ありません。本アプリの利用により生じた損害について、制作者は一切の責任を負いません。
